(Q-7)How can image repetition of the backup be controlled?
--> to control the repetition of an image in the background, use the background-repeat property.
--> you cam use no-repeat value for the background-repeat property.
--> the image will display only once.
--> EX:-
body{
    background-image: url("image path")
    background-repeat: repeat;
}
(Q-8)What is the use of background-position property?
--> the background-position property sets the starting position of a background image.
--> background-image is placed at the top-left corner of an element and repeated both vertically and horizontally.
--> EX:-
body{
    background-image: url("image path")
    background-repeat: no-repeat;
    background-position: 30% 60%
}
(Q-9) Which property controls the image scroll in the background? --> the background-attachment property sets a background image scrolls with the rest of the page,or is fixed.
--> EX:-
body{
    background-image: url("image path")
    background-repeat: no-repeat;
    background-attachment: scroll;
}

(Q-16) What are the advantages and disadvantages of external style sheets?
1) advantages
--> one change to the style sheet will change all linked pages.
--> you can create classes of style that can then be used on many different html element.
2) disadvantages
--> your pages may not be rendered correctly until the external css is loaded
--> uploading or linking to multiple css files can increase your site download time.
