jQuery Basic, Effects & Advanced

(Q-1) What is jQuery?
--> jquery is a fast ,small, cross-platform and feature- rich javascript library.
--> it is designed to simplify the client-side scripting of HTML.
--> it is also used to add animation.

(Q-2) How to Apply CSS Using JQuery, How to Add Class and Remove Class in Jquery , JQuery Animation?
1) apply jQuery
--> the css() method sets or returns one or more style properties for the selected element.
--> to return the value of a specified css property.
-->EX:-
$("p").css("background-color");
2) add class
--> the addClass() method adds one or more class names to the selected element.
-->Ex:-
$("button").click(function(){
$("p:first").addClass("intro");
});
3) jQuery addclass()
--> adds one or more classes to the selected elements.
--> Ex:-
$("button").click(function(){
  $("#div1").addClass("important blue");
});
4) jQuery Animation
--> the animate() method performs a custom animation of a set of css properties.
--> this method changes an element from one state to another with css styles.
-->EX:-
$("button").click(function(){
   $("#box").animate({height:"300px"});
});
